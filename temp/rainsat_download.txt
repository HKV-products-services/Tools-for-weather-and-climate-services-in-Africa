This ZIP file contains the Rainsat export data

Depending on the input settings this file contains:

- statistics_{name}.json
- rainsat_{name}.nc
- rainsat_download.txt

The netCDF file georeferencing is done with the EPSG:4326 projection and contains precipitation in milimeter per hour.

All times are in the UTC timezone
